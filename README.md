# May 10 Project 🌸

Este repositório é dedicado ao desenvolvimento de um projeto especial e pessoal focado em criar uma experiência digital personalizada (nesse caso, foi projetado como um dos presentes de aniversário de um ano de namoro). O projeto utiliza tecnologias web fundamentais para construir uma interface interativa e visualmente atraente.

## 🚀 Sobre o Projeto

Este projeto foi pensado para ser:

- **Interativo:** Utilizando JavaScript para criar respostas em tempo real.
- **Escalável:** Código organizado para permitir a adição fácil de novas páginas e funções.
- **Personalizável:** O estilo está sendo refinado para oferecer uma experiência única.

## 🛠️ Tecnologias Utilizadas

- **HTML:** Estrutura semântica dos conteúdos;
- **CSS:** Estilização, layout e responsividade;
- **JavaScript:** Interatividade e dinamismo;
- Podem ser adicionadas novas tecnologias ao longo do projeto.
